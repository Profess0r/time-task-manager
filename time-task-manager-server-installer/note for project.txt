создание установщика TimeTaskanager

программа установки должна предусматривтаь следующие шаги

1. просит админа указать путь куда развернуть сервер и копирует его в эту папку
2. просит админа создать пользователя который обладает правами создавать базу
данных
3. вводим данные ip-адрес порт юзер нэйм пароль пользователя
4. накатываем под этим пользователем скрипт на базу данных
5. прописываем данные для соединения в пропертис в папке где установлен сервер
6. запускаем сервер.

в шагах/плэйнах

1. Первый плэйн - сообщение о том что будет произведена установка серверной части приложения 
    сообщение о том что уже необходимо чтобы были установлены одна из трех БД и были права 
    для создания пользователя БД.
    
2. Просим ввести пользователя путь для копирования туда файла приложения сервера.
   Если такого пути нет - создаем его.
   
3. Просим пользователя выбрать одну из трех баз данных.

4. Просим пользователя создать в базе данных пользователя с соответствующими правами предлагая ему пример скрипта
   в соответствии с ранее выбранной БД или же предлагаем ему ввести данные админа и создать автоматически пользователя 
   из установщика.
   
5. Предлагаем пользователю ввести IP - адрес сервера на котором находитс БД, номер порта (предлагая варианты по умолчанию), 
   имя пользователя и его пароль. Если пользователя создавали мы сами то мы жестко прописываем имя пользовател в поле имя пользователя 
   и запрещаем его менять.
   
6. Показываем юзеру скрипт который будет накатываться на БД. После нажатия на next накатываем скрипт, если все ок 
   то создаем файл с пропертисами в папке где установлено приложения сервера.

7. Поздравляем пользователя с установкой и предлагаем опционально запустить сервер и выйти из приложения.
